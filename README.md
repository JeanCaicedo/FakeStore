# FakeStore

Ejercicio de primeros pasos con JavaScript vanilla: tienda web que consume la Fake Store API, con catálogo, detalle de producto, carrito, favoritos, login y confirmación de compra.

## Contexto

Práctica de autoestudio para armar un e-commerce básico con varias vistas usando HTML, CSS y JavaScript vanilla dividido en módulos (`api.js`, `auth.js`, `cart.js`, `favorites.js`, etc.). Se practica consumo de API, navegación entre páginas y manejo de estado con `localStorage`.

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla, módulos)
- [Fake Store API](https://fakestoreapi.com/)

## Ejecución

```bash
# Abrir index.html en el navegador
# o servirlo con cualquier servidor estático:
npx serve .
```
